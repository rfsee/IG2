# IG Auto Growth OS - Execution Roadmap v2

## Phase 1 - Data and Measurement Foundation (Week 1)
- Add post-level metric fields and daily metric records.
- Add funnel KPI cards in frontend.
- Add baseline report export.

### Exit Criteria
- Every published post has reach/save/dm/click/order data.
- Dashboard can show save_rate, dm_rate, click_rate, order_rate.

## Phase 2 - Content + Visual Automation (Week 2)
- Content Agent generates caption blocks and CTA variants.
- Visual Prompt Agent generates one prompt per script requirement block.
- Add copy-ready outputs in post editor.

### Exit Criteria
- Editor outputs are one-click copyable.
- Prompt count equals script requirement count.

## Phase 3 - A/B Experiment Engine (Week 3)
- Add experiment table for each post.
- Auto-generate variant B from baseline.
- Auto-declare winner from 7-day KPI thresholds.

### Exit Criteria
- At least one A/B test per week.
- Winner is written back to next content suggestions.

## Phase 4 - DM Conversion Engine (Week 4)
- Add DM intent classifier and script templates.
- Add product recommendation snippets for DM.
- Track thread stages to closed.

### Exit Criteria
- DM scripts generated by intent.
- Conversion stages measurable from new -> closed.

## Phase 5 - Growth Analyst Loop (Week 5)
- Weekly auto-summary of winners/losers.
- Auto-generate next-week strategy and task list.

### Exit Criteria
- Weekly report created from real data.
- Next-week plan includes trigger mix and A/B hypotheses.

## Agent Work Allocation
- `Content Agent`: hooks, caption, CTA variants, trigger tags.
- `Visual Prompt Agent`: image prompt blocks from script steps.
- `Publishing Agent`: status transitions and preflight checks.
- `Engagement Agent`: DM/comment scripts and product routing.
- `Growth Analyst Agent`: metrics diagnostics and next actions.

## Operational Rhythm
- Daily: update post status, ingest metrics, handle DM scripts.
- Weekly: review A/B outcomes, lock winners, generate next week plan.
- Monthly: prune low-performing formats, raise winning pattern share.
